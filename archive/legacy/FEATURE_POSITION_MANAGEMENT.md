- ❌ **没有主动减仓**已有持仓
- ❌ 市场从低位涨上来，仓位被动累积到 93%

**之前逻辑**:
```
情绪高涨 (64 分) → 建议仓位 60%
  ↓
过滤新买入信号 ✅
不主动减仓已有持仓 ❌
  ↓
实际仓位：93% (被动维持)
```

## 修复方案

### ✅ 新增主动仓位管理

**Phase 2.5: 情绪指数主动仓位管理** (新增)

在策略信号前，先检查当前仓位：

```python
# 计算当前仓位
current_position_pct = 持仓市值 / 总资产 * 100

# 检查是否超过情绪上限
if current_position_pct > 情绪上限 + 10%:
    主动减仓至目标仓位
```

**减仓逻辑**:
1. 优先减仓**亏损/盈利少**的股票
2. 每只减仓**30%**
3. 最多减**5 只**
4. 直到接近目标仓位

## 减仓优先级

**优先减仓顺序**:
1. **亏损股票** (优先)
2. **盈利少的股票** (<5%)
3. **盈利中等的股票** (5-10%)
4. **盈利多的股票** (>10%，最后)

**示例**:
```

4. 工业富联：+2.3% (盈利少，减 30%)

预期效果：
```
