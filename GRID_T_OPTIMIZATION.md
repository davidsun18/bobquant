# 📈 做 T 策略优化方案

## 🐛 当前问题

**固定 3% 就高抛 → 卖飞强势股！**

例如：
- 某股票今天强势上涨 8%
- 3% 时卖出了
- 后面继续涨到 8%
- 卖飞了 5% 的涨幅！❌

---

## ✅ 优化方案

### 1️⃣ 趋势判断 (不做 T 的情况)

**强势股不做 T**:
```python
# 股价在 5 日均线上方 2% 以上
if current > ma5 * 1.02:
    return 0, ''  # 强势上涨，不做 T

# RSI < 65 (未超买)
if rsi < 65:
    return 0, ''  # 未超买，不做 T
```

**只在震荡市做 T**:
- 股价在均线附近震荡 → ✅ 做 T
- 股价强势上涨 → ❌ 不做 T
- 股价破位下跌 → ❌ 不做 T

---

### 2️⃣ 动态阈值

根据股票走势调整做 T 阈值：

| 走势 | 触发阈值 | 说明 |
|------|----------|------|
| **强势上涨** | 5% | 涨得多才卖，避免卖飞 |
| **震荡市** | 3% | 正常做 T |
| **下跌趋势** | 2% | 早点跑，少亏点 |

---

### 3️⃣ 分批做 T

不一次性卖完，分批高抛：

```
第 1 次：涨 3% 卖 30%
第 2 次：涨 5% 卖 30%
第 3 次：涨 7% 卖 40%
```

这样即使卖飞一部分，还有仓位享受上涨！

---

### 4️⃣ 移动止盈

**卖出后股价继续涨**:
```
卖出价：¥50
现价：¥52 (继续涨)
→ 不接回！让利润奔跑
→ 等创新高后回落 2% 再接回
```
