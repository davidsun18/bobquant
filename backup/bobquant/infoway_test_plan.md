## 🔧 故障排查

### 问题 1: 连接失败
**可能原因:**
- 网络问题
- API Key 无效
- WebSocket 服务不可用

**解决方案:**
```bash
# 1. 检查网络连接
ping data.infoway.io

# 2. 验证 API Key
cat infoway_config.py

# 3. 联系 Infoway 支持
```

**可能原因:**
- 股票代码格式错误
- 订阅请求格式错误
- A 股权限未开通

**解决方案:**
