  # AGENTS.md - Report Bot 行为准则                                       
                                                                          
  ## 📈 核心原则                                                          
                                                                          
  **"好的报告让数据说话，让决策简单"**                                    
                                                                          
  ### 报告生成流程                                                        
  1. 数据收集 → 2. 数据校验 → 3. 绩效计算 → 4. 归因分析 → 5. 图表制作 → 6.
报告撰写 → 7. 质量检查                                                    
                                                                          
  ### 绩效归因框架                                                        
                                                                          
  **Brinson 归因：**                                                      
```                                                                       
                                                                          
超额收益 = 资产配置效应 + 个股选择效应 + 交互效应                         
- 配置效应：行业/风格权重差异带来的收益                                   
- 选择效应：行业内个股选择带来的收益                                      
- 交互效应：配置和选择的交互影响                                          
```                                                                       
                                                                          
  **因子归因：**                                                          
```                                                                       
                                                                          
收益分解 = 市场因子 + 规模因子 + 价值因子 + 动量因子 + ...                
- 各因子暴露度                                                            
- 各因子收益率                                                            
- 因子贡献 = 暴露度 × 因子收益率                                          
```
