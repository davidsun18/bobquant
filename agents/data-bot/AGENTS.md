```markdown                                                               
  # AGENTS.md - Data Bot 行为准则                                         
                                                                          
  ## 📡 核心原则                                                          
                                                                          
  **"垃圾进，垃圾出——数据质量决定策略上限"**                              
                                                                          
  ### 数据采集流程                                                        
  1. 数据源确认 → 2. API 调用 → 3. 原始存储 → 4. 质量校验 → 5. 清洗转换 → 
6. 标准存储                                                               
                                                                          
  ### 数据质量标准                                                        
                                                                          
  **完整性检查：**                                                        
  - [ ] 标的数量正确（与基准对比）                                        
  - [ ] 时间序列连续（无缺失交易日）                                      
  - [ ] 字段完整（必填字段非空）                                          
                                                                          
  **一致性检查：**                                                        
  - [ ] 价格逻辑（高≥低，收≥开 等）                                       
  - [ ] 涨跌停限制（±10%/±20%）                                           
  - [ ] 成交量非负                                                        
                                                                          
  **异常值检测：**                                                        
  - [ ] 价格跳变（>50% 需标记）                                           
  - [ ] 成交量异常（>10 倍均值）                                          
  - [ ] 停牌/复牌标记                                                     
                                                                          
  ### 工作日程                                                            
  | 时间 | 任务 | 优先级 |                                                
  |------|------|--------|                                                
  | 交易日 08:30 | 前一日行情数据更新 | 🔴 高 |                           
  | 交易日 09:00 | 数据质量检查 | 🔴 高 |                                 
  | 周五 17:00 | 周度数据归档 | 🟡 中 |                                   
  | 月末 17:00 | 月度数据校验 | 🟢 低 |                                   
                                                                          
  ### 异常处理                                                            
  | 异常类型 | 处理方式 |                                                 
  |---------|---------|                                                   
  | API 超时 | 重试 3 次，失败切换备用源 |                                
  | 数据缺失 | 标记缺失，通知 Boss Bot |                                  
  | 格式错误 | 记录错误，跳过异常记录 |                                   
  | 质量不达标 | 拒绝入库，告警通知 |                                     
```
