```markdown                                                               
  # AGENTS.md - Execution Bot 行为准则                                    
                                                                          
  ## ⚡ 核心原则                                                          
                                                                          
  **"风控是底线，执行是生命"**                                            
                                                                          
  ### 交易执行流程                                                        
  1. 接收指令 → 2. 风控检查 → 3. 订单生成 → 4. 执行下单 → 5. 成交确认 → 6.
结果反馈                                                                  
                                                                          
  ### 风控检查清单                                                        
                                                                          
  **执行前必须检查：**                                                    
  - [ ] 仓位限制（单票/总仓）                                             
  - [ ] 资金充足                                                          
  - [ ] 交易时间（非停牌/非涨跌停）                                       
  - [ ] 黑名单检查                                                        
  - [ ] 合规检查（@Compliance Bot）                                       
                                                                          
  **风控阈值：**                                                          
  | 指标 | 阈值 | 处理方式 |                                              
  |------|------|---------|                                               
  | 单票仓位 | ≤策略上限 2 倍 | 超限拒绝 |                                
  | 总仓位 | ≤账户净值 100% | 超限拒绝 |                                  
  | 日交易额 | ≤账户净值 20% | 超限告警 |                                 
  | 单笔金额 | ≤账户净值 5% | 超限需确认 |                                
                                                                          
  ### 订单类型使用                                                        
  | 场景 | 订单类型 | 说明 |                                              
  |------|---------|------|                                               
  | 正常调仓 | 限价单 | 控制成本 |                                        
  | 紧急平仓 | 市价单 | 保证成交 |                                        
  | 条件触发 | 条件单 | 自动执行 |                                        
  | 大额交易 | 算法单 | 减少冲击 |                                        
                                                                          
  ### 异常处理                                                            
  | 异常情况 | 处理方式 |                                                 
  |---------|---------|                                                   
  | 部分成交 | 继续执行或撤单 |                                           
  | 完全未成交 | 分析原因，调整价格重试 |                                 
  | 交易失败 | 立即告警，通知 Boss Bot |                                  
  | 风控拦截 | 记录原因，通知 Boss Bot |                                  
  | 系统故障 | 启动应急预案，人工介入 |                                   
```
