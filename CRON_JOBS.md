```markdown                                                               
  # CRON_JOBS.md - Agent 定时任务配置                                     
                                                                          
  ## ⏰ 定时任务清单                                                      
                                                                          
  ### 每日任务                                                            
                                                                          
  | 时间 | 执行 Agent | 任务 | 优先级 |                                   
  |------|-----------|------|----- ---|                                   
  | 08:30 | Data Bot | 启动数据采集（前一日行情） | 🔴 高 |               
  | 09:00 | Execution Bot | 检查开盘前状态 | 🔴 高 |                      
  | 11:30 | Data Bot | 午间数据备份 | 🟡 中 |                             
  | 15:00 | Execution Bot | 收盘前持仓检查 | 🔴 高 |                      
  | 15:30 | Execution Bot | 生成交易日报告 | 🟡 中 |                      
  | 16:00 | Data Bot | 盘后数据归档 | 🟡 中 |                             
  | 17:00 | Report Bot | 生成绩效日报 | 🟢 低 |                           
                                                                          
  ### 每周任务                                                            
                                                                          
  | 时间 | 执行 Agent | 任务 | 优先级 |                                   
  |------|-----------|------|----- ---|                                   
  | 周一 09:00 | Quant Research Bot | 生成周度策略回顾 | 🟡 中 |          
  | 周五 17:00 | Data Bot | 周度数据校验 | 🟡 中 |                        
  | 周五 18:00 | Report Bot | 生成周度绩效报告 | 🟢 低 |                  
                                                                          
  ### 每月任务                                                            
                                                                          
  | 时间 | 执行 Agent | 任务 | 优先级 |                                   
  |------|-----------|------|----- ---|                                   
  | 1 日 10:00 | Compliance Bot | 生成月度合规报告 | 🔴 高 |              
  | 1 日 14:00 | Quant Research Bot | 月度因子回顾 | 🟡 中 |              
  | 1 日 16:00 | Report Bot | 生成月度绩效报告 | 🔴 高 |                  
  | 月末 17:00 | Data Bot | 月度数据归档 | 🟡 中 |                        
                                                                          
  ## 📋 Cron 表达式                                                       
                                                                          
  ```bash                                                                 
  # 每日任务                                                              
  30 8 * * *    # Data Bot 数据采集                                       
  0 9 * * *     # Execution Bot 开盘检查                                  
  30 15 * * *   # Execution Bot 交易报告                                  
  0 17 * * *    # Report Bot 日报                                         
                                                                          
  # 每周任务                                                              
  0 9 * * 1     # Quant Research Bot 周度回顾                             
  0 18 * * 5    # Report Bot 周报                                         
                                                                          
  # 每月任务                                                              
  0 10 1 * *    # Compliance Bot 月度合规                                 
  0 16 1 * *    # Report Bot 月报                                         
```                                                                       
                                                                          
🔔 通知规则                                                               
                                                                          
- 高优先级任务 - 完成后立即通知 Boss Bot                                  
- 中优先级任务 - 完成后记录日志                                           
- 低优先级任务 - 批量汇总通知                                             
                                                                          
⚠️ 异常处理                                                               
                                                                          
- 任务执行失败 → 重试 3 次 → 告警通知                                     
- 任务超时 → 记录原因 → 通知 Boss Bot                                     
- 数据异常 → 暂停后续任务 → 人工介入                                      
```
