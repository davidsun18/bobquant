## 🚀 使用方法

### 快速开始

```bash
cd /home/openclaw/.openclaw/workspace/quant_strategies/bobquant

# 1. 创建配置文件
mkdir -p config
cat > config/bobquant.json5 << 'EOF'
{
  system: { mode: "simulation", log_level: "INFO" },
  account: { initial_capital: 1000000 },
  data: { primary: "tencent" },
  stock_pool: [
    { code: "000001.SZ", name: "平安银行", strategy: "dual_macd" },
  ],
}
EOF

# 2. 运行主程序
python3 -m bobquant.main
```

### 编程方式使用

```python
from bobquant.main import BobQuantEngine
from pathlib import Path

# 创建引擎
engine = BobQuantEngine(config_path=Path("config/bobquant.json5"))

# 初始化
if not engine.initialize():
    exit(1)

# 执行一次检查
trades = engine.run_check()

# 查看账户汇总
summary = engine.portfolio_summary()

# 关闭
engine.shutdown()
```

## ⚠️ 已知问题

### 1. 规则匹配器 API
**问题**: `RuleMatcher.add_rule()` 期望 `Rule` 对象而非字典  
**影响**: 测试脚本失败，不影响主程序  
**解决**: 使用 `Rule` 对象或更新 API

### 2. AuditLogger API
**问题**: `AuditLogger.log()` 需要 `tool_name` 和 `tool_use_id` 参数  
**影响**: 测试脚本失败，不影响主程序  
**解决**: 传入必需参数或更新 API

### 3. BatchConfig API
**问题**: 参数名不匹配（应为 `max_batch_size` 而非 `batch_size`）  
**影响**: 测试脚本失败，不影响主程序  
**解决**: 使用正确的参数名

### 4. 相对导入
**问题**: main.py 使用相对导入，直接运行会失败  
**影响**: 需要以模块方式运行  
**解决**: 使用 `python3 -m bobquant.main`
