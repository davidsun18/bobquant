### 2. ✅ 实现 5 层配置继承

**层级结构**:
1. **Global Defaults** (`GlobalDefaults`) - 全局默认配置
2. **Per-Strategy** (`PerStrategyConfig`) - 策略级配置
3. **Per-Channel** (`PerChannelConfig`) - 渠道级配置
4. **Per-Account** (`PerAccountConfig`) - 账户级配置
5. **Per-Group** (`PerGroupConfig`) - 组级配置

**优先级**: Per-Group > Per-Account > Per-Channel > Per-Strategy > Global Defaults

**实现方法**:
- `BobQuantConfig.resolve()` 方法逐层合并配置
- 使用 `deep_merge()` 深度合并字典
- 支持通过 `active_strategy/channel/account/group` 激活特定配置

**示例**:
```json5
{
  "global_defaults": {
    "account": {"initial_capital": 1000000}
  },
  "account_configs": {
    "account_001": {"account": {"initial_capital": 500000}}
  },
  "active_account": "account_001"
}
// 最终 initial_capital = 500000 (账户级覆盖)
```

### 4. ✅ 实现 SecretRef（环境变量/文件/命令）

**实现类**: `SecretRef`

**支持的类型**:
```python
class SecretType(Enum):
    ENV = "env"    # 环境变量
    FILE = "file"  # 文件
    CMD = "cmd"    # 命令
```

**使用方式**:
```python
# 编程方式
SecretRef(type=SecretType.ENV, ref="API_KEY")

# JSON5 字符串格式
"${env:API_KEY}"
"${file:~/.secret.txt}"
"${cmd:vault read key}"
```

**解析方法**:
- `SecretRef.resolve()` - 解析为实际值
- `ConfigLoader.load_with_secrets()` - 加载并自动解析所有 SecretRef

### 5. ✅ 支持 JSON5（注释 + 尾随逗号 + 环境变量替换）

**实现**:
- 使用 `json5` 库解析 JSON5 格式
- `ConfigLoader.load_json5()` 方法支持：
  - 注释（`//` 和 `/* */`）
  - 尾随逗号
  - 环境变量替换（`${VAR}` 和 `$VAR`）

**示例**:
```json5
{
  // 这是注释
  "system": {
    "name": "BobQuant",
    "version": "3.0",  // 尾随逗号
  },
  "account": {
    "api_key": "${env:BOBQUANT_API_KEY}"  // 环境变量替换
  }
}
```

### 6. ✅ 实现配置验证和错误处理

**实现类**: `ConfigValidator`

**验证类型**:
1. **Schema 验证** - Pydantic 自动验证
2. **业务规则验证** - 自定义验证逻辑
   - 仓位比例范围检查
   - 止损比例符号检查
   - 交易时段逻辑检查
3. **SecretRef 验证** - 检查是否可解析

**错误处理**:
- `ValidationError` 自定义异常类
- 详细错误报告：`validator.get_error_report()`
- 错误包含字段名和建议

**使用示例**:
```python
validator = ConfigValidator(config)
if not validator.validate_all():
    print(validator.get_error_report())
    # 输出：
    # 配置验证失败，发现以下问题：
    # 1. 单票最大仓位比例必须在 0-1 之间 (字段：account.max_position_pct)
    #    建议：例如：0.10 表示 10%
```

## 使用示例

### 加载配置

```python
from bobquant.config import ConfigLoader, ConfigValidator

# 加载配置
loader = ConfigLoader("config/settings.json5")
config = loader.load(
    strategy="bollinger_conservative",
    channel="sim_channel",
    account="account_001"
)

# 验证配置
validator = ConfigValidator(config)
if not validator.validate_all():
    print(validator.get_error_report())
    exit(1)

# 解析 SecretRef
config = config.resolve_secrets()

# 使用配置
print(f"初始资金：{config.account.initial_capital}")
print(f"最大仓位：{config.position.max_position_pct}")
```

### 配置迁移

```python
from bobquant.config.migrations import migrate_config

# 迁移配置文件
target, records = migrate_config(
    Path("config/settings_v2.2.json5"),
    backup=True
)

# 查看迁移记录
for record in records:
    status = "✓" if record.success else "✗"
    print(f"{status} {record.source_version} -> {record.target_version}")
```
