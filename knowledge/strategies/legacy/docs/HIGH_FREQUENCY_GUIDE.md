## 🚀 核心特性

### 1. 超高频交易
- **检查间隔**: 3 秒（原 5 秒）
- **触发阈值**: 0.15%（原 0.3%）
- **最小盈利**: 1 个价位即止盈
- **最长持仓**: 5 分钟（时间止损）

### 2. 四大高频策略

| 策略 | 触发条件 | 持仓时间 | 目标收益 | 风险 |
|------|---------|---------|---------|------|
| **剥头皮** | MACD 金叉 + RSI 极端 | 1-5 分钟 | 0.2% | 低 |
| **动量交易** | 2 分钟涨跌≥0.2% + 放量 | 2-10 分钟 | 0.4% | 中 |
| **均值回归** | 偏离均线 0.2% | 5-15 分钟 | 0.2% | 低 |
| **突破交易** | 突破 5 日高点 + 放量 | 10-30 分钟 | 2% | 高 |

### 3. 激进参数配置

| 参数 | 保守版 | 激进版 | 变化 |
|------|--------|--------|------|
| 做 T 触发 | 2% | 0.15% | **13 倍** |
| 止盈阈值 | 5% | 2% | 2.5 倍 |
| 止损阈值 | -8% | -5% | 收紧 |
| 持仓上限 | 10 只 | 15 只 | +50% |
| 单票仓位 | 10% | 15% | +50% |

## 📋 启用方法

### 1. 修改配置文件

编辑 `bobquant/config/sim_config_v2_4_aggressive.yaml`:

```yaml
# 启用高频模式
high_frequency:
  enabled: true                   # ✅ 开启
  max_trades_per_minute: 5        # 每分钟最多 5 笔
  min_spread: 0.001               # 最小价差 0.1%

# 策略配置
strategy:
  day_trading:
    enabled: true
    t_grid_up: 0.0015             # 0.15% 触发
    t_grid_down: 0.0015
    scalping_threshold: 0.001     # 剥头皮阈值
```

### 2. 启动模拟盘

```bash
cd /home/openclaw/.openclaw/workspace/quant_strategies
python3 -m bobquant.main --config config/sim_config_v2_4_aggressive.yaml
```

### 3. 监控高频交易

查看实时日志：
```bash
tail -f logs/sim_trading_highfreq/*.log
```

## ⚠️ 风险提示

### 1. 高频交易风险
- **手续费累积**: 日交易 50 笔，手续费约 0.25%/日
- **滑点损失**: 快速进出可能遇到较大滑点
- **过拟合风险**: 策略可能在特定市场有效

### 2. 适合的市场环境
- ✅ **震荡市**: 均值回归策略表现最佳
- ✅ **温和上涨**: 剥头皮 + 动量策略
- ❌ **单边暴跌**: 建议关闭高频，降低仓位
- ❌ **流动性差**: 避免小盘股，选择龙头股

### 3. 建议仓位管理
- **新手**: 先用 10% 资金测试
- **有经验**: 50% 资金
- **激进**: 满仓（需密切监控）

## 🔧 参数调优

### 剥头皮策略
```yaml
scalping:
  threshold: 0.001        # 0.1% 触发，可调 0.0005-0.002
  max_holding_time: 300   # 5 分钟，可调 180-600 秒
  min_profit_tick: 1      # 1 个价位，可调 1-3
```

### 动量策略
```yaml
momentum:
  threshold: 0.002        # 0.2% 动量，可调 0.001-0.005
  volume_confirm: 1.2     # 量比 1.2 倍，可调 1.0-2.0
```

### 均值回归
```yaml
mean_reversion:
  threshold: 0.002        # 偏离 0.2%，可调 0.001-0.005
  ma_period: 20           # 20 日均线，可调 10-50
```

## 📞 常见问题

**Q: 为什么我的高频策略不赚钱？**
A: 可能原因：
- 手续费过高（检查佣金率）
- 滑点过大（避免流动性差的股票）
- 参数不适合当前市场（需要调优）

**Q: 一天最多交易多少笔？**
A: 建议上限 50-100 笔，过多会增加手续费成本

**Q: 适合哪些股票？**
A: 高流动性龙头股：
- 银行：平安银行、招商银行
- 白酒：贵州茅台、五粮液
- 科技：宁德时代、比亚迪

**Q: 需要实时监控吗？**
A: 是的，高频交易需要密切监控，建议：
- 设置飞书实时通知
- 设置最大亏损自动停止
- 定期检查持仓状态
