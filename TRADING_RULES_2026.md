## 📅 一、交易时间

### 交易日
- **日期**: 每周一至周五 (国家法定节假日除外)
- **调休**: 法定节假日调休工作日正常交易

### 交易时段

| 时段 | 时间 | 交易机制 | 说明 |
|------|------|----------|------|
| 开盘集合竞价 | 09:15 — 09:25 | 集中撮合 | 确定开盘价 |
| 申报暂存 | 09:25 — 09:30 | 暂存 | 可申报，暂存券商系统 |
| 早盘连续竞价 | 09:30 — 11:30 | 逐笔撮合 | 连续交易 |
| 午休 | 11:30 — 13:00 | 暂存 | 可申报，暂存券商系统 |
| 午盘连续竞价 | 13:00 — 14:57 | 逐笔撮合 | 连续交易 |
| 收盘集合竞价 | 14:57 — 15:00 | 集中撮合 | 确定收盘价 |
| 盘后固定价格 | 15:05 — 15:30 | 固定价 | 科创板/创业板 (拟扩展至全部 A 股) |

**注意**:
- 09:25-09:30 和 11:30-13:00 期间可提交申报，但申报暂存于证券公司系统
- 待进入交易时段后 (09:30 或 13:00) 方上报至交易所主机

## 🎯 二、竞价与成交规则

### 基本原则
**"价格优先、时间优先"**

#### 价格优先
- 较高价格的**买入申报**优先于较低价格的买入申报
- 较低价格的**卖出申报**优先于较高价格的卖出申报

#### 时间优先
- 买卖方向、价格均相同时，先申报者优先成交
- 先后顺序以**交易所主机接收申报的时间**为准

### 撮合方式

| 方式 | 时段 | 特点 |
|------|------|------|
| 集合竞价 | 09:15-09:25, 14:57-15:00 | 集中撮合，产生开盘/收盘价 |
| 连续竞价 | 09:30-11:30, 13:00-14:57 | 逐笔连续撮合 |

## 📦 三、T+1 交收制度

### 股票交收
- **T 日买入**: 当天买入的股票，所有权在 T+1 日完成过户
- **T 日可卖**: 当天买入的股票**无法在当天卖出**

### 资金交收
- **T 日卖出**: 所得资金当天可用于继续买入其他股票 (**T+0 可用**)
- **T+1 可取**: 卖出所得资金需到 T+1 日才能转出至银行账户 (**T+1 可取**)

### 制度目的
抑制过度投机，维护市场稳定

## 📏 四、交易单位

### 普通 A 股 (主板、创业板)
- **买入**: 100 股 (1 手) 的整数倍
- **卖出**: 余额不足 100 股时须一次性卖出
- **单笔最大**: 不超过 100 万股

### 科创板
- **买入**: 200 股起步，200 股整数倍
- **卖出**: 余额不足 200 股时须一次性卖出
- **单笔最大**: 不超过 100 万股

### 示例
```
✓ 正确：100 股、200 股、1000 股 (主板)
✓ 正确：200 股、400 股、2000 股 (科创板)
✗ 错误：50 股、150 股、250 股
```

## 💰 六、交易费用

### 收费标准 (2026 年最新)

| 费用项目 | 收费标准 | 收取方式 | 说明 |
|----------|----------|----------|------|
| **佣金** | ≤3‰，最低 5 元/笔 | 买卖双向 | 券商可协商，主流万 1-万 3 |
| **印花税** | 0.5‰ (万分之五) | 仅卖出单边 | 国家统一 |
| **过户费** | 0.01‰ (十万分之一) | 买卖双向 | 国家统一 |

### 费用计算示例

**买入 100 股 @ ¥100.00 (佣金万三)**:
```
成交金额：100 × 100 = ¥10,000.00
佣金：10,000 × 0.0003 = ¥3.00 → 按最低¥5.00 收取
过户费：10,000 × 0.00001 = ¥0.10
印花税：¥0.00 (买入不收)
总费用：¥5.10
```

**卖出 100 股 @ ¥100.00**:
```
成交金额：100 × 100 = ¥10,000.00
佣金：10,000 × 0.0003 = ¥3.00 → 按最低¥5.00 收取
过户费：10,000 × 0.00001 = ¥0.10
印花税：10,000 × 0.0005 = ¥5.00
总费用：¥10.10
```

### 费率对比

| 券商类型 | 佣金率 | 万元交易佣金 |
|----------|--------|--------------|
| 高费率 | 3‰ | ¥30.00 |
| 主流费率 | 1-3‱ | ¥10-30 |
| 低费率 | 1‱ | ¥10.00 |

## 🌙 七、盘后固定价格交易

### 已实施

## ⚠️ 九、短线交易限制

### 适用对象
- 上市公司持股 5% 以上大股东
- 董事、监事、高级管理人员

### 限制内容
- **6 个月内**: 买入后又卖出 (或卖出后又买入)
- **收益归属**: 所得收益归公司所有

### 实施日期
2026 年 4 月 7 日起施行

### 目的
防范内幕交易和短线操纵

## 🚨 十、退市整理期与风险警示

### 退市整理期
- **首日**: 无涨跌幅限制
- **后续**: ±10% (主板/创业板/科创板)

### 风险警示股票 (ST/*ST)
- **涨跌幅**: ±5% (拟调整±10%)
- **买入限制**: 部分板块设单日买入上限 (北交所征求意见稿)

### 风险提示
ST/*ST 股票存在较高风险，投资者需谨慎

## 🔧 框架实现

### TradingRules 类

```python
from framework.trading_rules import TradingRules, TradingConfig

rules = TradingRules(TradingConfig(
    commission_rate=0.0003,  # 万三
    stamp_duty=0.0005,       # 万分之五
    transfer_fee=0.00001     # 十万分之一
))
```

### 主要方法

| 方法 | 功能 |
|------|------|
| `can_trade()` | 检查是否可交易 |
| `validate_order()` | 验证订单合法性 |
| `calculate_cost()` | 计算交易费用 |
| `check_price_limit()` | 检查涨跌停 |
| `get_board_type()` | 判断股票板块 |
| `record_buy()` | 记录买入 (T+1) |
| `get_available_quantity()` | 计算可卖数量 |

### 使用示例

```python
# 检查交易时间
can, reason = rules.time_controller.can_trade()
print(f"可交易：{can} ({reason})")

# 验证订单
valid, error = rules.validate_order(
    stock_code="600519",
    action="buy",
    price=1400.00,
    quantity=100
)

# 计算费用
cost = rules.calculate_cost(1400.00, 100, is_buy=True)
print(f"总费用：¥{cost['total_cost']:.2f}")

# T+1 规则
rules.record_buy("600519", 100)
available = rules.get_available_quantity("600519", 200)
print(f"可卖：{available} 股")
```
