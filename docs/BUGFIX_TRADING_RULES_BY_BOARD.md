## 📊 不同板块规则

| 板块 | 代码特征 | 最小买入 | 申报步长 | 单笔上限 (限价) |
|------|----------|----------|----------|----------------|
| **主板** | 0/6 开头 | 100 股 | 100 股 | 100 万股 |
| **创业板** | 30 开头 | 100 股 | 100 股 | 30 万股 |
| **科创板** | 688 开头 | 200 股 | 1 股 | 10 万股 |

**零股规则**:
- 不足 100 股 (科创板 200 股) 的零股必须**一次性卖出**
- 不能拆分申报

## 🐛 原代码问题

### 问题 1: 科创板买入错误
```python
# 原代码
shares = int(shares / 100) * 100  # 全部按 100 股整数倍

# 问题：科创板 200 股起！
# 例如：计算得出 150 股 → 规范化为 100 股 ❌ (应为 200 股)
```

### 问题 2: 科创板申报错误
```python
# 原代码
shares = int(shares / 100) * 100

# 问题：科创板超过 200 股可以 1 股递增！
# 例如：201 股 → 规范化为 200 股 ❌ (201 股是合法的)
```

### 问题 3: 零股卖出错误
```python
# 原代码
if shares < 100:
    continue  # 跳过不卖

# 问题：零股必须一次性卖出！
# 例如：持有 50 股 → 应该全部卖出，而不是跳过
```
