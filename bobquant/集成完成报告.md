## 🎯 功能说明

### 核心功能
1. **实时数据展示**: 自动从账户文件和交易记录加载数据
2. **图表可视化**: 使用 Plotly 创建交互式图表
3. **多页面导航**: 侧边栏菜单，5 个独立页面
4. **数据筛选**: 支持按股票、操作类型筛选
5. **实时刷新**: 30 秒自动缓存更新，支持手动刷新
6. **响应式设计**: 适配不同屏幕尺寸

### 技术亮点
- 使用 `@st.cache_data(ttl=30)` 实现智能缓存
- Plotly 交互式图表（缩放、平移、悬停）
- Pandas 数据处理和格式化
- 腾讯财经 API 实时行情
- 侧边栏导航和状态显示

## 🚀 使用方法

### 启动 Streamlit
```bash
cd /home/openclaw/.openclaw/workspace/quant_strategies/bobquant
./start_streamlit.sh
```

### 启动导航页
```bash
cd /home/openclaw/.openclaw/workspace/quant_strategies/bobquant
./start_nav_page.sh
```

### 后台运行（已执行）
```bash
nohup streamlit run web/streamlit_app.py \
  --server.address 0.0.0.0 \
  --server.port 8501 \
  --server.headless true > /tmp/streamlit.log 2>&1 &
```

### 查看状态
```bash
# 检查进程
ps aux | grep streamlit

# 查看日志
tail -f /tmp/streamlit.log

# 健康检查
curl http://localhost:8501/_stcore/health
```

### 停止服务
```bash
pkill -f "streamlit run"
```
