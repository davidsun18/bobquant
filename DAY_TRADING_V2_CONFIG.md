## 🎯 策略参数

### 趋势跟踪
```python
MA_FAST: 20     # 快线
MA_SLOW: 60     # 慢线
买入：MA20 金叉 MA60
卖出：MA20 死叉 MA60
```

### RSI 过滤
```python
RSI_PERIOD: 14
RSI_BUY_MAX: 40    # RSI<40 买入
RSI_SELL_MIN: 70   # RSI>70 卖出
```

### 成交量确认
```python
VOLUME_RATIO: 1.5  # 成交量>1.5 倍均量
```

### 移动止盈
```python
激活条件：盈利>20%
止盈条件：从高点回撤 10%
```

### 移动止损
```python
初始止损：-15%
盈利>10% 后：止损上移至成本价
盈利>20% 后：止损上移至 +10%
```

### 暴跌买入 ⭐
```python
触发条件：大盘跌破 60 日线 5%

第 1 天：买入 30% 仓位
第 2 天：继续暴跌→买入 20% 仓位
第 3 天：继续暴跌→买入 10% 仓位

总计：60% 仓位
```

### 金字塔补仓
```python
-8%:  补 50% 仓位
-15%: 补 30% 仓位
```

### 动态仓位
```python
根据历史胜率调整:
  胜率>60%: 20% 仓位
  胜率>50%: 15% 仓位
  胜率<50%: 10% 仓位
```

### 持仓限制
```python
最小持仓：5 只
最大持仓：8 只
单只最大：30%
总仓位：≤90% (暴跌时)
现金储备：≥10%
```
