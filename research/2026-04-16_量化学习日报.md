## 四、改进方案：动态止损模块设计

### 方案 A：ATR 动态止损（推荐优先实现）

```python
def calculate_dynamic_stop_loss(entry_price, atr, multiplier=2.5):
    """
    基于 ATR 的动态止损
    
    参数:
        entry_price: 入场价格
        atr: 平均真实波幅（14 日）
        multiplier: ATR 倍数（通常 2-3）
    
    返回:
        stop_loss_price: 止损价格
    """
    stop_loss = entry_price - (multiplier * atr)
    return max(stop_loss, entry_price * 0.92)  # 最多不超过 8% 固定止损
```

**优势**：
- 自动适应股票波动率（高波动股票止损宽，低波动股票止损窄）
- 避免被正常波动震出
- 业界标准做法

### 方案 B：追踪止损（Trailing Stop）

```python
def update_trailing_stop(current_price, highest_price_since_entry, trailing_pct=0.05):
    """
    追踪止损：价格创新高时自动上移止损点
    
    参数:
        current_price: 当前价格
        highest_price_since_entry: 入场后最高价
        trailing_pct: 回撤比例（默认 5%）

        trailing_stop: 追踪止损价

    trailing_stop = highest_price_since_entry * (1 - trailing_pct)
```

## 五、改进方案：波动率自适应仓位

### 核心思路

```python
def calculate_volatility_adjusted_position(
    account_value,
    risk_per_trade=0.02,      # 每笔交易风险 2%
    entry_price,
    stop_loss_price,
    volatility_adjustment=1.0  # 波动率调整系数
):
    """
    波动率自适应仓位计算
    
    公式：
    position_size = (account_value * risk_per_trade * volatility_adjustment) / (entry_price - stop_loss_price)
    
    波动率调整系数：
    - 低波动（ATR/Price < 2%）：volatility_adjustment = 1.2（加仓）
    - 中波动（2%-4%）：volatility_adjustment = 1.0（标准）
    - 高波动（>4%）：volatility_adjustment = 0.7（减仓）
    """
    risk_distance = entry_price - stop_loss_price
    position_value = (account_value * risk_per_trade * volatility_adjustment) / risk_distance
    return position_value
```

```python
class EnhancedRiskManager(RiskManager):
    """增强版风险管理器"""
    
    def check_order_with_dynamic_stop(self, order, position):
        """
        带动态止损的订单检查
        """
        # 1. 计算 ATR 动态止损点
        atr = self.get_atr(order.symbol, period=14)
        dynamic_stop = order.price - (2.5 * atr)
        
        # 2. 计算基于止损的仓位
        risk_distance = order.price - dynamic_stop
        max_position = (self.current_value * 0.02) / risk_distance
        
        # 3. 检查是否超过动态仓位限制
        if order.quantity > max_position:
            return RiskCheckResult.FAIL, f"超过动态仓位限制：{max_position}股"
        
        # 4. 检查流动性（订单量 < 日均成交量 5%）
        avg_volume = self.get_avg_volume(order.symbol, days=20)
        if order.quantity > avg_volume * 0.05:
            return RiskCheckResult.WARNING, f"订单量超过日均成交量 5%"
        
        return RiskCheckResult.PASS, "风控检查通过"
    
    def update_trailing_stops(self):
        """
        更新所有持仓的追踪止损点
        """
        for symbol, position in self.positions.items():
            current_price = self.prices.get(symbol, 0)
            
            # 更新最高价
            if not hasattr(position, 'highest_price'):
                position.highest_price = current_price
            else:
                position.highest_price = max(position.highest_price, current_price)
            
            # 计算追踪止损（从最高点回撤 5%）
            position.trailing_stop = position.highest_price * 0.95
            
            # 如果跌破追踪止损，生成卖出信号
            if current_price < position.trailing_stop:
                self.generate_exit_signal(symbol, 'trailing_stop')
```
