### 1. ⭐⭐⭐ vnpy (VeighNa) - Python 量化交易平台
**GitHub**: https://github.com/vnpy/vnpy

**核心思路**: 
- 国内最成熟的开源量化交易框架，4.0 版本新增 vnpy.alpha 模块，专为 AI 量化策略设计
- 内置 Alpha 158 因子库（源自微软 Qlib），支持 Lasso/LightGBM/MLP 等多种机器学习算法
- 提供完整的投研工作流：数据管理 → 模型训练 → 信号生成 → 策略回测

**对我们的帮助**:

### 2. ⭐⭐⭐ QUANTAXIS - 分布式量化解决方案
**GitHub**: https://github.com/yutiansut/QUANTAXIS

**核心思路**:
- 全新 v2.1 版本集成 QARS2 Rust 核心，实现 100x 账户操作加速（50ms→0.5ms）
- 10x 回测速度提升，90% 内存优化，支持 Python/Rust/C++ 跨语言 QIFI 协议
- 内置因子研究模块、多账户管理、分布式计算 agent

**对我们的帮助**:
- Rust 高性能核心思路值得参考，关键路径可用 Rust 重写提升性能
- QIFI 统一账户协议设计优秀，可借鉴用于规范我们的账户/持仓数据结构
- 共享内存数据传输方案（QADataBridge）对实时行情分发有参考价值

### 3. ⭐⭐⭐ Microsoft Qlib - AI 量化投资平台
**GitHub**: https://github.com/microsoft/qlib

**核心思路**:
- 微软研究院出品，AI 驱动的量化投研平台，支持监督学习/强化学习等多种建模范式
- 最新推出 RD-Agent：LLM 驱动的自主量化研发 Agent，可自动挖掘因子和优化模型
- 提供完整的 ML 工作流：特征工程 → 模型训练 → 组合优化 → 回测分析

**对我们的帮助**:
- RD-Agent 的 LLM 自动因子挖掘思路前沿，可探索将 AI 用于策略研发自动化
- 特征工程数据集（dataset）模块设计完善，可参考构建我们的因子库
- 模型标准化 API 设计（统一接口切换不同算法）值得借鉴
