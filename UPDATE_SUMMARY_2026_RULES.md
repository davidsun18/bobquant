### 2. 交易费用标准 (最新)

| 费用 | 旧标准 | **新标准** |
|------|--------|-----------|
| 佣金 | 万五 (0.05%) | **万三 (0.03%)**, 最低 5 元 |
| 印花税 | 0.1% (千分之一) | **0.05% (万分之五)** |
| 过户费 | 0.002% | **0.001% (十万分之一)** |

**费用计算示例**:
```
买入 1000 股 @ ¥50.00:
  成交金额：¥50,000
  佣金：¥15.00 (万三)
  过户费：¥0.50
  总费用：¥15.50 (费率 3.1‱)

卖出 1000 股 @ ¥50.00:
  成交金额：¥50,000
  佣金：¥15.00
  过户费：¥0.50
  印花税：¥25.00 (万分之五)
  总费用：¥40.50 (费率 8.1‱)
```

### 3. 板块交易单位

| 板块 | 最小买入 | 数量步长 |
|------|----------|----------|
| 主板/创业板 | **100 股** | 100 股 |
| **科创板** | **200 股** | 200 股 |

**验证规则**:
```python
# 科创板买入验证
validate_order('688001', 'buy', 50.00, 100)  # ✗ 科创板最小买入 200 股
validate_order('688001', 'buy', 50.00, 200)  # ✓ 通过
validate_order('688001', 'buy', 50.00, 300)  # ✗ 须为 200 股整数倍
```

### 4. 涨跌停限制

| 板块 | 涨跌幅 | 备注 |
|------|--------|------|
| 沪深主板 | ±10% | 标准 |
| ST/*ST | ±5% | **拟调整±10%** (征求意见) |
| 科创板 | ±20% | - |
| 创业板 | ±20% | - |
| **北交所** | **±30%** | 新增支持 |

**临时停牌规则** (无涨跌幅限制股票):
- 较开盘价±30% → 临停 10 分钟
- 较开盘价±60% → 临停 10 分钟

### 6. 订单验证增强

**新增验证项**:
- ✅ 科创板 200 股起步验证
- ✅ 板块涨跌幅限制验证
- ✅ 单笔最大申报 (100 万股)
- ✅ 卖出时不足 100 股须一次性卖出
- ✅ 申报暂存时段允许提交

**验证示例**:
```python
# 主板订单
valid, reason = rules.validate_order('600519', 'buy', 1400.00, 100)
# → True, "验证通过"

# 科创板订单 (100 股)
valid, reason = rules.validate_order('688001', 'buy', 50.00, 100)
# → False, "科创板最小买入 200 股"

# 科创板订单 (200 股)
valid, reason = rules.validate_order('688001', 'buy', 50.00, 200)
# → True, "验证通过"
```

## 🚀 使用指南

### 快速开始

```python
from framework.trading_rules import TradingRules, TradingConfig

# 初始化 (使用最新默认配置)
rules = TradingRules(TradingConfig())

# 检查交易时间
can, reason = rules.time_controller.can_trade()

# 验证订单
valid, error = rules.validate_order(
    stock_code='600519',
    action='buy',
    price=1400.00,
    quantity=100
)

# 计算费用
cost = rules.calculate_cost(1400.00, 100, is_buy=True)
print(f"总费用：¥{cost['total_cost_no_slippage']:.2f}")
```
