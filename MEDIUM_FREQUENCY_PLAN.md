## 🎯 设计目标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| **交易频率** | 5-30 分钟 | 基于 5/15/30 分钟 K 线 |
| **持仓周期** | 1-5 天 | 短线波段，非日内 |
| **延迟要求** | <500ms | 腾讯财经分钟线足够 |
| **策略类型** | 多策略并行 | 网格 + 波段 + 动量 |
| **最大回撤** | <15% | 严格风控 |

## 📈 策略类型

### 1️⃣ 分钟级网格交易
**周期**: 5 分钟 K 线  
**逻辑**: 
- 价格每波动 1% 触发一次买卖
- 网格间距：0.5%-1%
- 单次仓位：2-5%

**适用股票**: 波动率高的科技股、半导体

---

### 2️⃣ 日内波段策略
**周期**: 15 分钟 K 线  
**逻辑**:
- MACD 金叉 + RSI<30 → 买入
- MACD 死叉 + RSI>70 → 卖出
- 持仓时间：1-3 天

**适用股票**: 趋势明显的白酒、银行

---

### 3️⃣ 动量突破策略
**周期**: 30 分钟 K 线  
**逻辑**:
- 突破 20 周期高点 → 买入
- 跌破 20 周期低点 → 卖出
- 配合成交量确认

**适用股票**: 强势股、龙头股

## 🔄 交易流程

```
┌─────────────────────────────────────────┐
│  定时轮询 (每 5 分钟)                    │
│  交易时段：09:30-11:30, 13:00-15:00    │
└───────────────┬─────────────────────────┘
                │
                ▼
┌─────────────────────────────────────────┐
│  1. 获取分钟线数据                       │
│     - 腾讯财经：5/15/30 分钟 K 线          │
│     - 缓存：最近 100 根 K 线               │
└───────────────┬─────────────────────────┘
                │
                ▼
┌─────────────────────────────────────────┐
│  2. 计算技术指标                         │
│     - MACD, RSI, KDJ                    │
│     - 布林带，均线系统                   │
│     - 成交量分析                         │
└───────────────┬─────────────────────────┘
                │
                ▼
┌─────────────────────────────────────────┐
│  3. 生成交易信号                         │
│     - 网格：价格突破网格线               │
│     - 波段：指标金叉/死叉                │
│     - 动量：突破高低点                   │
└───────────────┬─────────────────────────┘
                │
                ▼
┌─────────────────────────────────────────┐
│  4. 风控检查                             │
│     - 仓位限制：单只≤10%                │
│     - 日内交易次数：≤5 次/只             │
│     - 止损：-3% 硬止损                    │
└───────────────┬─────────────────────────┘
                │
                ▼
┌─────────────────────────────────────────┐
│  5. 执行交易                             │
│     - 记录交易日志                       │
│     - 更新持仓数据                       │
│     - 发送通知 (可选)                    │
└─────────────────────────────────────────┘
```

## ⚙️ 配置参数

### 基础配置
```yaml
medium_frequency:
  # 时间配置
  check_interval: 300          # 检查间隔 (秒) = 5 分钟
  trading_hours:
    morning: "09:30-11:30"
    afternoon: "13:00-15:00"
  
  # 数据配置
  kline_periods: [5, 15, 30]   # K 线周期 (分钟)
  kline_limit: 100             # 获取 K 线数量
  
  # 股票池
  stock_pool:
    - sh.603986  # 兆易创新
    - sz.002371  # 北方华创
    - sz.300750  # 宁德时代
    # ... 10-15 只高波动股票
```

### 策略配置
```yaml
strategies:
  # 网格策略
  grid:
    enabled: true
    grid_size: 0.01            # 网格间距 1%
    position_per_grid: 0.03    # 每格仓位 3%
    max_grids: 10              # 最多 10 格
    
  # 波段策略
  swing:
    enabled: true
    rsi_oversold: 30           # RSI 超卖
    rsi_overbought: 70         # RSI 超买
    macd_fast: 12
    macd_slow: 26
    macd_signal: 9
    
  # 动量策略
  momentum:
    enabled: true
    breakout_period: 20        # 突破周期
    volume_confirm: 1.5        # 成交量确认倍数
```

### 风控配置
```yaml
risk_control:
  # 仓位限制
  max_position_per_stock: 0.10   # 单只≤10%
  max_total_position: 0.60       # 总仓位≤60%
  min_cash_reserve: 0.40         # 现金≥40%
  
  # 交易限制
  max_trades_per_day: 5          # 每日最多 5 笔/只
  min_trade_interval: 300        # 最小间隔 5 分钟
  
  # 止损止盈
  stop_loss: -0.03               # -3% 止损
  take_profit: 0.08              # +8% 止盈
  trailing_stop: true            # 移动止盈
  
  # 异常检测
  max_consecutive_losses: 3      # 连续亏损 3 笔暂停
  max_daily_loss: -0.05          # 日亏损>-5%
```

## ⚠️ 风险提示

1. **数据延迟**: 腾讯财经非实时数据，可能有 100-200ms 延迟
2. **交易成本**: 频繁交易手续费累积
3. **滑点风险**: 实际成交价可能与信号价有偏差
4. **过拟合**: 参数优化可能过拟合历史数据
